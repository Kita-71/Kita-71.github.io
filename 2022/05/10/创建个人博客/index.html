Windows 操作系统下的操作

初期环境搭建

  1. 安装 nodejs
    Download | Node.js (nodejs.org)
  2. 安装 hexo
  1. 创建仓库
    • 命名为 Kita-71.github.io
    • 配置好 git(基本操作，不详细展开)
  2. Hexo 初始化
    • 工作目录使用 Terminal 运行 Hexo init 进行初始化
      ![[创建个人博客/Pasted image 20220510170107.png]]
    • 终端运行 Hexo 测试
      • hexo clean： 清理编译文件
      • hexo g：编译项目
      • hexo s(server) : 本地预览运行项目（浏览器访问以下地址可以看到网页）
        ![](创建个人博客/Pasted image 20220510170229.png)
        ![](创建个人博客/Pasted image 20220510170022.png)
