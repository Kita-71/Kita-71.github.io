Windows 操作系统下的操作

初期环境搭建

  1. 安装 nodejs
    Download | Node.js (nodejs.org)
  2. 安装 hexo
  1. 创建仓库
    • 命名为 Kita-71.github.io
    • 配置好 git(基本操作，不详细展开)
  2. Hexo 初始化
    • 工作目录使用 Terminal 运行 Hexo init 进行初始化
    • 终端运行 Hexo 测试
      • hexo clean： 清理编译文件
      • hexo g：编译项目
      • hexo s(server) : 本地预览运行项目（浏览器访问其中的地址可以看到网页）

发布

  1. 修改 config.yml
  2. 安装 hexo-deployer-git 自动部署发布工具
  1. Hexo 编译上传
  1. Github 打开 Pages
